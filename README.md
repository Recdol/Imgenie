## 프로젝트 설명

<strong>IMGenie</strong>는 일상 이미지를 기반으로 어울리는 음악을 추천해주는 서비스입니다 🎧  
이미지를 게시하면 분위기, 상황, 날씨를 고려해 어울리는 노래를 추천해줍니다
